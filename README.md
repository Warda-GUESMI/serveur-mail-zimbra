# 📧 Serveur Mail Zimbra - DOCKER

> Mini projet 2ème année Informatique  
> Déploiement d'un serveur mail conteneurisé avec Docker

---

## 📋 Description

Ce projet consiste à déployer un serveur de messagerie **poste.io** (alternative Zimbra)  
dans un environnement **conteneurisé Docker**, permettant la gestion complète  
des comptes email, domaines virtuels et l'administration via une interface web.

---

## 🛠️ Technologies utilisées

- **Docker** & **Docker Compose**
- **poste.io** - Serveur mail complet
- **Linux** (CentOS / Rocky Linux)
- **SMTP / IMAP / POP3**

---

## ⚙️ Prérequis

- Docker installé
- Docker Compose installé
- Minimum 2 Go de RAM
- Port 25, 80, 443, 8443 disponibles
